✨ Key Features

🧩 Composable Skills

Each skill focuses on a single AL pattern or concern and can be loaded independently or combined with others.

🌍 Community-Driven

Contributors propose new skills via monthly release plans, get them approved, and submit implementations through Pull Requests.

📅 BC Update Alignment

Release plans follow the Business Central wave and cumulative update cadence so skills can target specific platform versions.

🤝 Agent-Agnostic

Skills work with any agent that supports the Agent Skills standard—GitHub Copilot, Claude Code, Cursor, and more.

🚀 Get Started

Using the Skills

  1. Browse available skills in the skills/ directory
  2. Copy a skill folder into your project's skill directory (e.g., .github/skills/ for GitHub Copilot)
  3. Invoke your agent — the skill will be loaded automatically when the task matches the skill's domain
